# 🌍 Kotlin Education Landscape Report

## Overview

This project explores how **Kotlin** is taught across universities, online learning platforms, and developer communities worldwide.

Kotlin, developed by JetBrains and officially supported by Google for Android development, has rapidly gained popularity among developers. As the language continues to grow in industry adoption, educational institutions and learning platforms have increasingly begun incorporating Kotlin into their programming curricula.

This project aims to analyze the **global Kotlin education landscape** by collecting and studying resources where Kotlin is taught.

---

## 🎯 Objectives

The main objectives of this project are:

- Identify universities and platforms teaching Kotlin
- Build a dataset of Kotlin learning resources
- Analyze trends in Kotlin education adoption
- Visualize how Kotlin learning resources are distributed across platforms
- Provide insights into how Kotlin is introduced to students and developers

---

## 📊 Dataset Overview

The dataset for this project contains information about Kotlin learning resources collected from multiple sources.

### Data Sources

The following sources were used to gather data:

- University course catalogs
- Online learning platforms (MOOCs)
- GitHub educational repositories
- Official Kotlin documentation and tutorials
- Developer education resources

Each dataset entry includes the following fields:

| Field | Description |
|------|-------------|
| Institution | Organization or platform offering the course |
| Country | Location of the institution or "Online" |
| Course Name | Name of the Kotlin learning resource |
| Category | Type of content (Android, Backend, Fundamentals, etc.) |
| Platform | Platform type (University, MOOC, GitHub, etc.) |
| Source | Link to the resource |

Currently, the dataset contains approximately **94 Kotlin learning resources**.

---

## 📈 Initial Analysis

Preliminary analysis of the dataset shows that Kotlin learning resources are distributed across multiple types of platforms.

### Platform Distribution

| Platform | Count |
|--------|-------|
| University | 32 |
| MOOC | 27 |
| GitHub | 16 |
| Training | 5 |
| Video | 5 |
| Official | 4 |

This distribution indicates that both universities and online platforms play a major role in teaching Kotlin.

---

## 🔍 Key Observations

Based on the current dataset, several observations can be made:

- Kotlin is increasingly being introduced in **university programming courses**.
- Online learning platforms provide widespread access to Kotlin education.
- GitHub repositories serve as practical learning resources for developers.
- A significant number of learning resources focus on **Android development**, reflecting Kotlin’s strong adoption in mobile development.
